To install node modules run:

For React 
npm i

For db access
cd backend
npm i

For Chat
cd Chat
npm i

For Video Chat
cd VideoChat
npm i


To run the code:

cd backend and type node server.js
cd Chat and type node server.js
cd VideoChat and type node server.js
start debugging the spring-boot api using "SPRING-BOOT DASHBOARD" which is a VS code extension
at the end cd to root folder and type npm start
